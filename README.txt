--Requerimientos--------------------

	Postgresql
	XAMP o cualquier otro servidor local

--Pasos-------------------
	
	1.- Clonamos el repositorio o en su defecto lo descargamos https://github.com/Is-Chelo/TallerdeEspecialidad
	
	2.- Ubicamos el proyecto en la carpeta htdocs -- C:\xampp\htdocs -- por lo general se encuentra en esa ruta
		
	3.- Creamos la base de datos en postgresql en este caso se uso PgAdmin, el nombre de la base de datos 
	sera dbtallerespecialidad

	4.- Restauramos la Base de Datos "dbtallerespecialidad.sql". esta accion de realiza dando click derecho en 
	la base de datos creada en la opcion Restore
	Si todo salio bien PgAdmin mostrara un mensaje de "completado con éxito" o "successfully completed"

.....   Genial! el proyecto esta Listo para ser usado 

NOTA: si desea cambiar el nombre de la base de datos considere que tambien debe cambiar en el archivo de
configuracion
